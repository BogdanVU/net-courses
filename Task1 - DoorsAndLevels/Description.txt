Создать консольное приложение которое выводит на экран 5 произвольных чисел включая 0. 
Пользователь может выбрать 0 либо любое другое число.
Если вводим число, которое отлично от 0, то попадаем на уровень N+1, где значения каждого числа равно [число на уровне N] * [выбранное число на уровне N];
Если вводим число 0, то попадаем на уровень N-1. 
Программа должна выводить числа на экран для каждого уровня.

Пример:
Даны числа 2 4 3 1 0 -> 
Выбираем число 2 -> 4 8 6 2 0 -> 
Выбираем число 4 -> 16 32 24 12 0 -> 
Выбираем число 0 -> 4 8 6 2 0 -> 
Выбираем число 0 -> 2 4 3 1 0

Попробую написать еще подробнее про задание номер 2:

Нам программа вывела числа:
2 4 3 1 0

Если мы вводим 2, то попадаем на второй уровень, на этом уровне числа умножаются на число которое ввели:

2*2 4*2 3*2 1*2 0*2 то есть
4 8 6 2 0

Далее если ввели например 4, то получаем

4*4 8*4 6*4 2*4 0*4
16 32 24 8 0

Если вводим ноль то опускаемся на предыдущий уровень
